# ReliefLink AI 🚨

## 🌍 Problem
During disasters, people struggle to find reliable information about nearby resources like food, shelter, and medical aid.

## 💡 Solution
ReliefLink AI is a web-based platform that maps real-time resources and helps users locate nearby help quickly.

## ⚙️ Features
- Map-based resource visualization
- Add new relief resources
- AI-based suggestion system
- Emergency help button
- 📍 Current location detection
